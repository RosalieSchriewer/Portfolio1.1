HTML og CSS

HTML og CSS er grunnmuren til enhver nettside.
HTML setter nettsides innhold på ønsket plass, mens CSS stiliserer innholdet.

Hva er HTML?

HTML eller «Hyper Text Markup Language» er standard språket til enhver nettside, oppfunnet i 1991. HTML forteller nettleseren hvordan innholdet skal vises.
HTML er brukt til å beskrive strukturen til nettsiden og består av flere elementer:

  • Overskrifter som signaliserer viktigheten av innholdet
  • Paragrafer som markerer hvor teksten starter
  • Pauser når det er ønsket en ny paragraf

HTML syntaks

Enhver HTML-element må være innfattet av forkortningen som er deskriptiv til resultatet. Dette gjør man ved bruk av vinkel parentes som ser slik ut <> . For å forklare det bedre tar vi for oss et paragraf element som skal ha innholdet «Dette er en paragraf»:
< p> Dette er en paragraf </p>
Som man kan se er altså paragrafen innfattet av en start og en slutt vinkel parentes.

Hva er CSS?

CSS er kort for «cascading style sheets». CSS ble først opfunnet i 1994 av norske informatikeren Håkon Wium Lie når han jobbet sammen med Tim Berners-Lee hos CERN.
Mens html er ansvarlig for nettsideoppsettet, så er CSS ansvarlig for å stilisere HTML-innholdet.
CSS er hva som gjør hver webside til en unik side.
Med CSS kan man sette en stemning på en nettside, tilpasset til ønsket publikum.
Det er mange små ting som gjør at en nettside blir til din nettside, og ved hjelp av CSS kan du endre:

  • Fargen
  • Font
  • Tekst størrelse
  • Posisjonen til elementer
  • Mellomrom mellom elementer
  • Bakgrunnsfarger og bakrgrunnsbilder
  • Skjermstørrelser og mye mer

«Cascading» mener i denne sammenhengen at stilen som ble satt til foreldre skal også bli anvendt på barna. Den «cascader» dermed over hele koden nedenfor.
CSS blir brukt i sammenheng med HTML. Man kan skrive css-en rett inn i html fila, eller linke til en ekstern CSS- fil som inneholder CSS-en.
Disse brukemetodene er kalt for «inline», «internal» og «external» CSS.

CSS syntaks

I CSS viser selektoren det HTML-elementet du vil endre stilen på. Det kan være en eller flere deklarasjoner i krøllparentes som må bli separert med et semikolon.
Hver deklarasjon skal ha et navn (property) som er knyttet til en verdi (value).
I eksemplet vist ovenfor er det en overskrift av gruppe 1 som skal være blå, med skriftstørrelse 12.
Viktig å nevne i sammenhengen med CSS-syntaksen er at den mest spesifikke reglelen alltid blir gjeldene. Hvis du for eksempel setter hele bodyen din til fargen blå, men spesifiserer at overskriften 1 skal være rød, så skal alltid overskriften blir rød, selv om den er satt i bodyen.
